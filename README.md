# Python Learning Repository

Structured Python learning materials, exercises and problem sets from coursework and self-study.

## Folder Structure

- **`00_theory`** – Notes, PDFs, and books (Python Crash Course, W3Schools summaries, teacher resources...).  
- **`01_ejercicios_clase`** – Python exercises completed in class.  
- **`02_relaciones_de_ejercicios`** – Structured problem sets and exercises by topic (types, loops, functions, lists, dictionaries...).

## Summary

This repository organizes Python learning into **theory**, **practice**, and **exercise collections**; providing a clear path from basics to applied problems.
